Welcome,

This site serves as a project archive and journal. It's intended to be useful to the reader and I hope it proves to be so over time.

It's also intended to be positive. There is enough negativity on social media already to last folks several lifetimes. I don't want that here. Nevertheless, I'd be remiss if I didn't make a few somewhat negative claims up front.

The modern programming landscape is fraught with issues. On the whole, quality is down significantly, and nearly all web solutions are slow, bloated, and over-engineered. We can do better.

This site strives to be a place for pragmatic, sensible ideas. Strives to be. I can always do better as well, but what I'd like to communicate is that there is still a need for craftsmanship, and that the software industry can be improved by a "do more with less" mindset.

I hope that you find your time here enjoyable and can take something worthwhile from this content.

All the best,
- Luke
